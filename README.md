# Python-Rock-Paper-Scissors
Project: Rock Paper Scissors — simple console game 
Run: python "Rock Paper Scissors Game.py"
Description: A tiny, interactive Rock–Paper–Scissors game for the terminal. Enter rock, paper, or scissors to play; q to quit.

Usage
Start the script: python "Rock Paper Scissors Game.py"
Input: Rock, Paper,Scissors, q = Quit
Outcomes: You vs computer, result printed each round.
